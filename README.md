# Memory Matching Game

A fun and interactive memory matching game built with jQuery where players flip cards to find matching pairs.

## Features

- Three difficulty levels (Easy, Medium, Hard)
- Move counter to track your performance
- Timer to challenge yourself
- Sound effects for matches and mismatches
- Responsive card flip animations
- Win message when all pairs are matched

## How to Play

1. Open `index.html` in your web browser
2. Select a difficulty level:
   - **Easy**: 4x2 grid (8 cards)
   - **Medium**: 4x4 grid (16 cards)
   - **Hard**: 6x4 grid (24 cards)
3. Click on cards to flip them and reveal symbols
4. Find matching pairs by remembering card positions
5. Match all pairs to win!

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery


## Installation

1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. Start playing!

No server or additional setup required.

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Game logic and functionality

## Game Rules

- Click any card to flip it
- Click a second card to find its match
- If cards match, they stay face up
- If cards don't match, they flip back after 1 second
- Game ends when all pairs are matched
